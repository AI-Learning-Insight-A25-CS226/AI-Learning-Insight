AI Learning Insight

**AI Learning Insight** adalah platform analisis pembelajaran berbasis AI yang membantu mendeteksi gaya belajar siswa berdasarkan data perilaku siswa dalam belajar. 
Proyek ini terdiri dari tiga bagian utama:
- **Machine Learning Service (FastAPI + Python)**
- **Backend API (Express + PostgreSQL)**
- **Frontend Dashboard (React + Vite)**

Struktur Proyek
AI-Learning-Insight/
├── ml-service/ # FastAPI model inference service
├── backend/ # Node.js RESTful API service
└── frontend/ # React dashboard app

Cara Menjalankan Program
Jalankan Machine Learning Service (FastAPI)
```
cd ml-service
python main.py
```
Jalankan Backend API (Node.js + Express)
```
cd backend
npm install
npm run db:migrate
npm run dev
```
Jalankan Frontend (React + Vite)
```
cd frontend
npm install
npm run dev
```
